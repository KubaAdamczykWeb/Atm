## Technologies
Project is created with:
* Node.js: 18.12.1
* Npm: 9.1.3
* React: 18.2.0
* Sass: 1.56.1
	
## Setup
To run this project, download and install using npm:
```
$ cd ../atm
$ npm install
$ npm start
```
You can setup currency, denominations and limits in file `/src/settings.js`
